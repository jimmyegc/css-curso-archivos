Crea el código completo en HTML y CSS para una landing page básica, atractiva y de alta conversión de una aplicación móvil. La página debe de ser limpia, moderna, responsiva y visualmente llamativa, diseñada para captar la atención y fomentar descargas en App Store y Google Play.

Instrucciones de estructura:

Sección Hero

Un encabezado convincente y conciso con la propuesta de valor de la app.

Un subtítulo breve que amplíe el beneficio principal

Dos botones de llamado a la acción (CTA) con enlaces a “App Store” y “Google Play”

Centra todo el texto y dale espacio superior e inferior suficiente para que se destaque respecto a la siguiente sección

Sección de Características/Beneficios

Una lista de 3 características principales, con descripciones cortas orientadas a beneficios

Cada característica acompañada de un ícono gratuito de font awesome

Sección de Prueba Social

Testimonios o reseñas breves y atractivas de usuarios

Mostrar la calificación promedio usando una estrella con los íconos gratis de font awesome

Sección de Llamado a la Acción

Otro bloque de botones de descarga iguales a los del Hero, que motive a descargar la app.

Principios de diseño:

Estilo limpio y minimalista, con espacios en blanco bien usados y uniformes

Diseños totalmente responsivos

Visuales de alta calidad, para íconos usar font awesome como recurso

Branding consistente, usando una paleta de colores moderna y coherente con la identidad de una aplicación digital

Entrega el resultado con el código HTML y CSS en archivos separados y bien comentados
