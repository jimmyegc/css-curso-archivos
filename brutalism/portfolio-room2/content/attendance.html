Event Management SaaS

Platform designed to manage large scale events including guest lists, badge generation, attendance tracking and seat organization.

React TypeScript Node.js PostgreSQL Tailwind

Key Features

Guest list management
Seat organization system
Badge generation with QR codes
Real-time attendance tracking
Admin dashboard
Event analytics reports

Architecture 


Frontend: React + TypeScript
Backend: Node.js + Express
Database: PostgreSQL
Infrastructure: AWS + NGINX
