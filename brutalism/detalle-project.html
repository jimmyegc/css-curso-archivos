IMAGEN DEL PROYECTO

Descripción

Este proyecto es un ejemplo de cómo aplicar el estilo neo-brutalista en interfaces modernas. Se utiliza tipografía bold, colores neon y bordes gruesos para generar impacto visual.

La estructura está diseñada para ser responsive y destacar la interacción con botones, animaciones de hover y efectos de sombras duras.

Tecnologías

  • HTML5 + Tailwind CSS
  • JavaScript ES6
  • Animaciones CSS / Hover Effects
  • Diseño Neo-Brutalista
  • Responsive Design
