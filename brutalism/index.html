Modal Brutalista

Este modal usa <dialog> con borde grueso y sombra dura.

Cómo personalizar

  1. Edita variables en :root para colores y grosor de borde.
  2. Duplica variantes de .btn cambiando --bg y color.
  3. Para inputs con error, agrega .error al contenedor .control y un .error-text debajo.
  4. El modal se maneja con JS usando HTMLDialogElement.
/* Ejemplo: cambiar el color primario */
:root{ --bui-primary: #EC5C2C; /* naranja nmda */ }
/* Bordes aún más brutales */
:root{ --bui-border-w: 4px; }
